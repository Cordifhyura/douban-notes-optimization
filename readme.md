## 豆瓣日记页功能优化
一个针对豆瓣日记页www.douban.com/note的小插件

把底BAR复制了一份放到了顶部

添加了一个目前为固定的小侧栏，能够显示标题和书影音并点击跳转

不同等级的标题设置了不同的大小和缩进，书影音按H2处理

(附加说明：编辑器里的标题1对应HTML标题的H2)
